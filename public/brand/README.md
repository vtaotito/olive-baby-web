# OlieCare Brand Assets

Pasta oficial de ativos da marca OlieCare.

## Arquivos Disponíveis

### SVGs (Vetores - Recomendado)
- `oliecare-logo-icon.svg` - Versão principal com fundo (512×512)
- `oliecare-logo-mark.svg` - Símbolo isolado (230×250)
- `oliecare-logo-horizontal.svg` - Versão completa com texto (420×120)
- `oliecare-logo-horizontal-dark.svg` - Versão para fundos escuros

### PNGs (pasta `png/`)
Gerados automaticamente em 9 tamanhos:
- **16px, 32px, 48px, 64px, 128px, 180px, 192px, 256px, 512px**
- Total: 27 arquivos PNG + favicon.ico

## Manual Completo

Abra o manual interativo no navegador:
```
public/brand-manual.html
```

## Como Gerar PNGs Novamente

```bash
node scripts/generate-brand-assets.mjs
```
